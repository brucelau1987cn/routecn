# Backtrace
- 仅支持Ubuntu/Debian
- 一个基于 NextTrace 的多回程线路自动化检查脚本。

## 功能
- 自动安装 NextTrace 引擎
- 支持电信、联通、移动、教育网等 12 个代表性节点
- 智能识别线路类型：CN2 GIA, CN2 GT, 联通9929, 移动CMIN2等
- 跨平台支持 (Linux/macOS)

## 使用方法
无需下载，直接运行：
```bash
curl -L https://raw.githubusercontent.com/brucelau1987cn/backtrace/main/backtrace.sh | bash
